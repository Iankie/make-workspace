## About
Script for creating dirs workspace for internal/external pentest project

## Installation 
Just clone the repo.
Recommend to create a symbolic link to /usr/bin/ dir.

## Usage 
- -m  - set mode
- -h  - print help
- -o  - open created workspace in VScode
  
#### Command  
```make-workspace -m (ext|int) [-o] [-h] /path/to/place```
